Skills

  • HTML (HyperText Markup Language) is the most widely used server-side language on the internet. It allows for the creation of the content and layout of web pages. HTML elements are used to format text, insert images, create links, and add other elements to the website.

  • CSS (Cascading Style Sheets) is a language used to control the appearance of web pages. CSS allows you to set the font, color, margin, layout, etc. of text. CSS files are referenced in HTML files, allowing you to specify the styles for the elements specified in the HTML file.

  • WebFlow is a web tool that allows for the quick and easy design and development of websites and applications. WebFlow is a visual web tool that enables users to create the layout and appearance of websites and applications using HTML, CSS, and JavaScript without having to deal with coding.
